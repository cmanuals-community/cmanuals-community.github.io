Examples

Here are some practical examples to help you understand C programming better:

Hello World Program


#include 

int main() {
    printf("Hello, World!\n");
    return 0;
}

Simple Addition


#include 

int main() {
    int a = 5, b = 10;
    int sum = a + b;
    printf("The sum is: %d\n", sum);
    return 0;
}

Using Loops


#include 

int main() {
    for (int i = 0; i < 5; i++) {
        printf("Count: %d\n", i);
    }
    return 0;
}
